# Discover: "Your list" reminder

**Revisit later:** Whether to add a **"your list"** (saved products per brand) inside Discover.

- **Watchlist** = items you’re watching with a **target price** (goal + alerts).
- **Your list** = saved **products per brand** you’re tracking for discovery (e.g. “these 5 Selkie dresses”) with no target required.

Right now Discover is **discovery-only**: Search + tracked brands → open topic → Buy / Sell opportunities. No per-brand saved product list. If you want “saved products I’m tracking for this brand” back, it can be added as a separate tab or section (e.g. “Your list” on the Discover [topic] screen) without changing Watchlist.
